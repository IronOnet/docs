Dremio
-------------

To connect to Dremio setup the URI as in this example :
```
dremio://<User>:<password>@<Host>:<Port-default-31010>/dremio
```

Where,

Host: Hostname or IP address of the coordinator
Port: ODBC port


If firewalls are implemented at the database level, then open the following Preset IP addresses to the database port:

* 35.161.45.11
* 52.32.136.34
* 54.244.23.85
